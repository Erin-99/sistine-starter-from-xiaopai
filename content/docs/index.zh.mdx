## 什么是 ClothCraft？

ClothCraft 是一款 **AI 虚拟试衣产品**，通过 Seedream 5.0 Lite 将模特照片与 1～3 件服装组合为完整穿搭。产品包含用户认证、订阅计费、每日次数限制、免费版自定义水印和中英文界面。

### 核心特性

- **虚拟试衣** — 将一张模特照片与最多三张服装图片组合成穿搭
- **Seedream 5.0 Lite** — 使用 Base64 图片输入生成 2K PNG
- **套餐限制** — 免费用户每天 3 次，付费用户每天 20 次
- **免费版水印** — 仅为免费用户的生成结果添加 ClothCraft 水印
- **用户认证** — 邮箱密码与可选的 Google OAuth（Better Auth）
- **订阅支付** — 通过 Creem 提供月付和年付
- **国际化** — 使用 next-intl 提供中英文界面

### 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | Next.js 16 (App Router) + React 19 |
| 样式 | Tailwind CSS + Framer Motion |
| 数据库 | PostgreSQL + Drizzle ORM |
| 认证 | Better Auth |
| 支付 | Creem |
| AI | 火山引擎（豆包 API） |
| 邮件 | Resend |
| 国际化 | next-intl |
