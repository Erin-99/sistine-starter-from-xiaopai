## What is ClothCraft?

ClothCraft is an **AI virtual try-on product** that combines a model photo with one to three garment images using Seedream 5.0 Lite. It includes authentication, subscription billing, daily usage limits, a custom free-plan watermark, and English and Chinese interfaces.

### Key Features

- **Virtual try-on** — Combine one model photo with up to three garment images
- **Seedream 5.0 Lite** — Generate a 2K PNG from Base64 image inputs
- **Plan limits** — Three daily try-ons for free users and twenty for paid users
- **Free watermark** — ClothCraft branding is applied only to free-plan results
- **Authentication** — Email/password and optional Google OAuth via Better Auth
- **Subscriptions** — Monthly and yearly checkout through Creem
- **Internationalization** — English and Chinese interfaces via next-intl

| Layer | Technology |
